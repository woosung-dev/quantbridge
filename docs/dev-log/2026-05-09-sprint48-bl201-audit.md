### 4.1 Caller trace monkeypatch test draft

`backend/tests/strategy/pine_v2/test_track_runner.py` 는 **dispatch_table 주입 spy** (TrackRunner 단독 테스트) 만 보유. `parse_and_run_v2` → `TrackRunner.invoke` **실제 호출 경로** 검증 테스트는 부재. LESSON-064 의무 (caller trace) 보강용 draft:

```python
# backend/tests/strategy/pine_v2/test_track_runner_caller_trace.py
# pine_v2 — LESSON-064 caller trace: parse_and_run_v2 가 실제로 TrackRunner.invoke 호출 (BL-201)
"""Sprint 48 P3 — BL-201 caller trace 검증 (audit recommendation).

LESSON-064 의무: dispatcher 통합 후 caller chain 이 monkeypatch 로 실측 가능해야
silent re-routing (예: 향후 누군가 compat.parse_and_run_v2 안에서 TrackRunner 우회
하고 직접 run_historical 호출) 을 회귀 테스트로 차단할 수 있다.
"""
from __future__ import annotations

from typing import Any

import pytest

from src.strategy.pine_v2 import compat
from src.strategy.pine_v2.track_runner import TrackRunner


@pytest.mark.asyncio
async def test_parse_and_run_v2_dispatches_to_track_runner(monkeypatch: pytest.MonkeyPatch) -> None:
    """parse_and_run_v2 호출 시 TrackRunner.invoke 가 실제로 발동.

    spy 가 발동되지 않으면 = 누군가 dispatcher 우회 = LESSON-064 위반.
    """
    called: dict[str, Any] = {"track": None, "kwargs": None, "count": 0}
    original_invoke = TrackRunner.invoke

    @classmethod
    def spy_invoke(cls: type[TrackRunner], track: str, **kwargs: Any) -> Any:
        called["track"] = track
        called["kwargs"] = kwargs
        called["count"] += 1
        return original_invoke(track=track, **kwargs)

    monkeypatch.setattr(TrackRunner, "invoke", spy_invoke)

    # 최소 fixture: Track S 로 분류되는 가장 단순한 Pine 소스 + dummy ohlcv.
    # (구체 fixture 는 P3 메인 세션이 기존 test_track_runner.py 의 fixture 재사용 결정)
    source = "//@version=5\nstrategy('test', overlay=false)\n"
    ohlcv = []  # P3 메인 세션이 deterministic fixture 로 교체

    try:
        compat.parse_and_run_v2(source=source, ohlcv=ohlcv)
    except Exception:
        # source 가 실제 strategy 가 아니어도 괜찮음 — invoke 가 호출됐는지만 검증.
        pass

    assert called["count"] >= 1, "TrackRunner.invoke 가 호출되지 않음 — dispatcher 우회 회귀"
    assert called["track"] in {"S", "A", "M"}, f"unexpected track: {called['track']!r}"
```

**상태:** draft only. Worker E 가 직접 commit 안 함. P3 메인 세션이 (a) fixture 결정 + (b) compat.parse_and_run_v2 의 minimal happy-path source 확정 후 commit 결정.

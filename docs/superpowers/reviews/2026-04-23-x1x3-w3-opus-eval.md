### 공통 hook 추출 권고: `useElementWidth`

**추출 후보:**

```tsx
// frontend/src/lib/hooks/use-element-width.ts (제안)
export function useElementWidth<T extends HTMLElement>(): {
  ref: React.RefObject<T | null>;
  hasWidth: boolean;
} {
  const ref = useRef<T | null>(null);
  const [hasWidth, setHasWidth] = useState(false);
  useEffect(() => {
    const node = ref.current;
    if (node === null) return;
    const initial = node.getBoundingClientRect().width;
    if (initial >= 1) {
      setHasWidth(true);
      return;
    }
    if (typeof ResizeObserver === "undefined") return;
    const observer = new ResizeObserver((entries) => {
      for (const entry of entries) {
        if (entry.contentRect.width >= 1) {
          setHasWidth(true);
          observer.disconnect();
          break;
        }
      }
    });
    observer.observe(node);
    return () => observer.disconnect();
  }, []);
  return { ref, hasWidth };
}
```

**권고 시점:** **지금은 미시급.** 두번째 차트 컴포넌트가 같은 패턴을 필요로 할 때 (Sprint X4+ 등) 추출. YAGNI 원칙 — 1 호출자만 있을 때 미리 추출하면 over-engineering.

**Sprint X1+X3 W3 범위 내 액션:** 없음. 추후 Sprint TODO.md 에 `[ ] hook 추출: useElementWidth (차트 컴포넌트 ≥ 2 시)` 추가만 권고.

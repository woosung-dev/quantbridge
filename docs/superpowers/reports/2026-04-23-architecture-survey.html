4. Whack-a-mole → Trust Layer 전환

Sprint Y1 (PR #61) 의 가장 큰 가치 — 사용자 신뢰도 + 엔지니어 생산성.

Before (X1+X3 dogfood)

1. backtest 실행 → FAILED
   "Call to 'ta.sar' not supported"
2. ta.sar 함수 추가 → PR merge
3. backtest 재실행 → FAILED
   "Call to 'ta.rma' not supported"
4. ta.rma 추가 → PR merge
5. backtest 재실행 → FAILED
   "Attribute access not supported: ta.tr"
6. ta.tr 추가 → PR merge
7. backtest 재실행 → FAILED
   "Call to 'fixnan' not supported"
8. ... (무한)

사용자 인내력 소진. 매 반복마다 docker rebuild + 5~10분 대기. "내가 돈 내고 쓸 품질" 미달.

After (Sprint Y1, PR #61)

1. strategy 등록
   → parse_preview 가 즉시 호출
   → analyze_coverage(source)
   → is_runnable + unsupported_builtins
2. UI 노란 경고 박스:
   "⚠ 미지원 (3건): fixnan,
    ta.supertrend, request.security"
3. 사용자가 backtest 시도:
   → 422 StrategyNotRunnable
   → 미지원 목록 + 문서 링크

백테스트 실행 전 에 명확. docs/02_domain/supported-indicators.md 화이트리스트로 사용자가 strategy 선택. Trust layer 철학 정합.

SSOT (Single Source of Truth) backend/src/strategy/pine_v2/coverage.py 의 SUPPORTED_FUNCTIONS + SUPPORTED_ATTRIBUTES. interpreter / stdlib 의 capability 와 SSOT 동기화 — 새 함수 추가 시 3 파일 (stdlib + interpreter + coverage) 갱신 필요.

Path β — dogfood + Tier-2 CI A 병행 + B (Tier-2 Trust Layer CI 자동화, ★) 1주 안전망 형성. dogfood 중 silent regression 방지 ★★★★★
Path γ — full reconnaissance A 병행 + ③ (TV 15-20 corpus 프로파일) + ② (PyneCore transformers 이식) 2주 Sprint 8b 의 정확한 scope 결정 + Tier-0 신뢰도 ↑ ★★★★☆
Path δ — bulk push A 병행 + ④ (top-N stdlib bulk 추가) 1~2주 indicator 커버리지 단기 ↑. 단, 우선순위 데이터 없이 진행 시 over-engineering 위험 ★★★☆☆
최종 권장: Path β

1주차: dogfood 시작 (i2_luxalgo loose 모드 + s1_pbr 등 qualified corpus). 동시에 Tier-2 CI 자동화 PR 진행 (~5일).
2주차: dogfood 피드백 종합 → Path γ (corpus 프로파일링) 또는 Path δ (bulk stdlib) 중 데이터 기반 선택.
3주차+: ADR-011 의 Sprint 8b 본 작업 (Tier-1 정제 + Tier-3 native).

왜 Path β 인가

  • Y1 의 가치 극대화 — Coverage Analyzer 는 사용자 facing. CI (Tier-2) 는 엔지니어 facing. 둘이 합쳐져야 trust layer 완성.
  • dogfood 와 시너지 — CI 가 매 commit 마다 silent regression 막아주므로 dogfood 안전성 ↑.
  • 비용 균형 — A 는 관찰 위주 (low effort), B 는 1주 작업. 합산 1주 + 관찰 1주.
  • 위험 분산 — γ/δ 는 우선순위 데이터 없이 진행 시 ROI 불명확. β 는 즉시 가치.

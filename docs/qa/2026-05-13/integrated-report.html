BL-244 상세 (재현 + Root Cause + Fix)

재현: Clerk 로그인 → POST /api/v1/optimizer/runs/grid-search (or /bayesian /genetic) → 500 Internal Server Error, text/plain, 14250 bytes

Root cause: slowapi/extension.py:382 _inject_headers 가 starlette.Response 인스턴스를 요구하지만 router 가 Pydantic OptimizationRunResponse 객체를 반환. FastAPI 의 Pydantic→JSONResponse 변환은 slowapi inject 뒤에 일어남 → slowapi 가 raw Pydantic 받아서 폭발.

Fix:

@router.post("/runs/grid-search", status_code=202, response_model=None)
@limiter.limit("5/minute")
async def submit_grid_search(request: Request, ...) -> JSONResponse:
    result = await service.submit_grid_search(...)
    return JSONResponse(result.model_dump(mode='json'), status_code=202)

Secondary fix: UnifiedErrorMiddleware (BL-163) 가 모든 500 도 {"detail": {"code": "internal_error", ...}} 정규화하여 다른 500 path 도 stack trace 누설 차단.

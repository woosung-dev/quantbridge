이 문서의 위치: 바이브·조합 A 방법론은 Level 1(Day 0) → Level 2(Sprint Plan) → Level 3(Task Superpowers 체인) → Level 2 복귀(Sprint 종료) 를 기술한다. 여기서 만드는 네 스킬은 2×2 매트릭스 (scope 탐색도 × sprint 묶음 수) 로 정리된다:
  • ① 수평 병렬 (양, scope 고정) — 작은 독립 sprint × N 동시
  • ② 수평 iteration + 배포 (scope 고정) — N 작은 sprint 주간 리듬 + canary/retro
  • ③ 수직 깊이 (1 대형, scope 탐색) — brainstorm 부터 PR 까지
  • ④ 수직 깊이 × iteration + 배포 (scope 탐색, 의존성 DAG) — N 깊은 sprint 체인 + iteration 배포
Level 1 Day 0 는 대상 아님 (1회성이라 자동화 이득 낮음).
업데이트 배경 (2026-04-20): Bundle 2 실측에서 스킬 1 이 pure UI 폴리시에만 잘 맞는다는 것이 확인. QuantBridge 의 실제 난이도 (Pine 엔진 확장 / Trading WebSocket / 새 도메인 모듈) 는 scope 가 미확정이라 스킬 1/2 로 커버 불가. 이를 위한 수직 깊이 스킬 두 개 (③ depth-sprint 단일 / ④ depth-chain iteration+배포) 를 추가. 상세 설계:
• autonomous-depth-sprint (스킬 3)
• autonomous-depth-chain (스킬 4)

네 스킬의 커버 범위:

  • ① autonomous-parallel-sprints (완료, Bundle 1/2 실측): /autoplan 산출 spec 을 받아 Level 3 체인을 N sprint 병렬 실행 + Evaluator 검증.
  • ② autonomous-sprint-cycle (미구현, H2 이후): 스킬 1 범위 + Sprint 종료 전체 (/qa · /canary · /retro+/learn) + 다음 Sprint spec 초안.
  • ③ autonomous-depth-sprint (설계 완료): raw idea 한 줄부터 brainstorming → autoplan 조건부 → writing-plans → executing-plans (TDD 정석) → Evaluator iter 5 → PR. 병렬 없음 (1 sprint 전용).
  • ④ autonomous-depth-chain (설계 완료): iteration goal → N depth sprint 의존성 DAG → 각 sprint 스킬 3 nested invoke → 각 완료 후 Integration Gate → iteration retro → Canary (30분). 의존성 체인 + 배포.

네 스킬 개요

① autonomous-parallel-sprints
(완료 · Bundle 2 dogfood)

축: 수평 병렬 (양)

입력: 확정 Sprint bundle spec

병렬도: 3~5

사람 개입: 최대 2회

자율 시간: 45~55분

적합: 작은 독립 UI 폴리시 × N

② autonomous-sprint-cycle
(미구현, H2 이후)

축: 수평 iteration + 배포

입력: iteration spec (bundle)

병렬도: N (스킬 1 상속)

사람 개입: 최대 4회

자율 시간: 60~120분

적합: 주간 iteration + staging→prod canary

③ autonomous-depth-sprint
(설계 완료, 프로토타입 대기)

축: 수직 깊이 (1 대형)

입력: raw idea / 1-liner 또는 plan

병렬도: 1

사람 개입: 최소 2회 (brainstorm + MR)

자율 시간: 2~6시간

적합: Pine 엔진 / WebSocket / 새 도메인

④ autonomous-depth-chain
(설계 완료, 스킬 3 이후 구현)

축: 수직 깊이 × iteration+배포

입력: iteration goal (2주짜리)

내부: 스킬 3 nested × N + DAG

사람 개입: N+2~3회 (iteration plan + 각 sprint MR + final MR + prod)

자율 시간: 8~20시간

적합: Pine v2 H2 심화 전체 / 엔진 레이어 iteration
